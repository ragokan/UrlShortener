# Url Shortener App

### You can check it online : https://shortur.vercel.app/

### Tecnologies

- React
- Firebase/firestore

### How to Use

- Rename 'ConfigForGithub' file to 'Config' in frontend/src/firebase.
- Add you firebase config to commented area.

### Start

- Type "yarn start" or "npm run start" to start.
